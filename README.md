# Python Document Scanner

这是一个基于 OpenCV 的简易文档扫描与矫正工具。它可以批量读取 `input` 目录下的图片，通过交互式界面让用户点击文档的四个角，自动进行透视变换（裁剪与矫正），并将处理后的图片保存到 `output` 目录。

## 功能特性

*   **批量处理**：自动遍历 `input` 文件夹中的所有图片（JPG, PNG, BMP, TIFF）。
*   **交互式裁剪**：在图片窗口中点击文档的四个角（顺序不限）。
*   **自动矫正**：根据选定的四个点，自动执行透视变换，拉直文档。
*   **视图控制**：支持缩放、平移图片，方便精准选点。
*   **双模式控制**：支持 **键盘快捷键** 和 **终端指令** 两种操作方式。
*   **非阻塞流程**：当前图片处理完成后自动切换到下一张，全部完成后自动退出。

## 依赖环境

需要安装 Python 3 以及以下库：

*   opencv-python
*   numpy

```bash
pip install opencv-python numpy
```

## 目录结构

确保你的项目目录结构如下：

```text
python-scanner/
├── input/          # 放入待扫描的原始图片
├── output/         # 处理结果会自动保存在这里
├── main.py         # 主程序
└── README.md
```

## 使用方法

1.  将需要扫描的图片放入 `input` 文件夹。
2.  运行主程序：

    ```bash
    python main.py
    ```

3.  程序会打开一个窗口显示第一张图片。
4.  **操作步骤**：
    *   在文档的四个角上分别点击鼠标左键（顺序不限）。
    *   选满 4 个点后，程序会自动裁剪并保存结果到 `output` 文件夹，然后自动加载下一张图片。

## 操作指南

你可以使用 **键盘快捷键** 或 **终端输入指令** 来控制视图。

### 常用快捷键 (并在窗口激活时)

| 按键 | 功能 |
| :--- | :--- |
| `+` / `=` | 放大视图 (Zoom In) |
| `-` | 缩小视图 (Zoom Out) |
| `W` | 向上移动 |
| `S` | 向下移动 |
| `A` | 向左移动 |
| `D` | 向右移动 |
| `Z` | 撤销上一个选点 (Undo) |
| `R` | 重置视图 (Reset) |
| `N` | 跳过当前图片 (Next) |
| `Q` | 退出程序 |

### 终端指令

在运行程序的终端中也可以输入以下指令并回车：

*   **缩放**：`zoom +` (放大), `zoom -` (缩小)
*   **移动**：`up`, `down`, `left`, `right`
*   **其他**：
    *   `undo`：撤销选点
    *   `reset`：重置视图位置和缩放
    *   `next`：跳过当前图片
    *   `quit` / `exit`：退出程序
    *   `help`：查看帮助

## 注意事项

*   如果图片分辨率很大，建议使用缩放功能 (`+` / `-`) 来调整视野以便更精确地点击角落。
*   输出的图片会保持与原图相同的文件名。
